# USSD Data Scratch Card Redemption System

A backend system for redeeming **data-only scratch cards via USSD**, designed
for Nigerian telecom infrastructure.

## Features
- USSD-based redemption
- Data-only scratch cards
- Multi-network support
- Secure, one-time PIN usage
- PostgreSQL-backed persistence
- Telco API integration
- Clean, extensible architecture

## Tech Stack
- Python 3.10+
- PostgreSQL
- SQLAlchemy
- Requests
- USSD Gateway compatible
